# Welcome to My Blog

This is an example blog post that demonstrates how the markdown content is rendered on your portfolio website.

## Features

- **Easy to manage**: Create, edit, and delete blog posts from the admin dashboard
- **GitHub integration**: All posts are stored and versioned in your GitHub repository
- **Clean design**: Posts are displayed beautifully on your blog page

## How to Get Started

1. Navigate to `/admin/login` to access the admin dashboard
2. Log in with your admin credentials
3. Click "Create New Post" to write a new blog post
4. Write your content in markdown format
5. Click "Save Post" to publish

## Markdown Support

You can use all standard markdown syntax:

- **Bold text** with `**text**`
- *Italic text* with `*text*`
- # Headings with `#`
- Lists with `-` or `*`
- And much more!
